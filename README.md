# _Loan Repayment Calculator_

##### by Rodnier Borrego Clavero

## Setup
* First, clone this repo:

```sh
$ git clone https://github.com/rodnierbc/loan-Repayment-Calculator.git
```
* Initialize NPM via  ```npm init```
* Install npm dependencies ```npm install```
* Run ```ng serve```
* Navigate to ```http://localhost:4200/```


## Technologies Used
* Node.js
* Angular 5
* Bootstrap 4.0.0
* typeScript
